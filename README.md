# pwa-text-editor
The challenge and task for this module was to build a PWA application that's in the form of a text editor. This challenge demonstrates the ability to create an application that not only runs natively on your web browswer but presents you with the option to download the application and have it run in native JavaScript.

## Installation <a name="installation"></a>

To install and run the Text Editor, follow these steps: <br>

1.  Clone the repo: git clone `<repository_url>`.
2.  Type in your terminal: `cd (cloned repo)`.
3.  Install the necessary dependencies: `npm install`.
4.  Start the application: `npm start`.
5.  Open your web browser and access the application at [http://localhost:3000](http://localhost:3000).
